## Question

#### Problem Description

For an input year N, find whether the year is a leap or not.

**Sample Input** 1
1600

**Sample Output** 1
true

**Explanation**
Year 1600 is a leap year

## Answer

- A year is a leap year if:(2 checks)
  - It is divisible by 400, OR
  - It is divisible by 4 but not divisible by 100.
- tc=1, sc=1
